Projects

Scrape Mega

This is a web scraper written in Python that looks for gaming laptops on the CanadaComputers website. After scraping the data, a new spreadsheet is generated with current prices, sales, and specifications such as RAM, CPU, and GPU of each laptop. I also made a website using Node.JS and Express, that allows for easy generation and viewing of the laptop specifications.

Automatic History Deleter

This is an extension written in Javascript, HTML, and CSS that automatically deletes history on Chrome startup, that includes certain blacklisted words as a substring, which can be entered in the options section of the chrome extension. There are also a few other functionalities that have been added like the ability to clear history within the last minute or last hour.

GPS Data Viewer and Modifier

This is a website that parses various .gpx files that contain GPS data through a C backend. The data can then be displayed and modified using the website made through Javascript, HTML, and CSS. SQL was also used to implement a database so that larger amounts of data can be used on the website.

Spooky Keylogger

This is a Windows program written in C that logs key presses and mouseclicks and then writes them to a log file that gets generated in the same directory, the program is hidden so it can only be found and closed through task manager, in the future I'm planning on adding the ability for the program to transmit the data to another machine through the internet.

Comic Book E-Store

This is a fully functional E-Store written in Java, that has a Swing GUI and was made using object-oriented programming practices to provide more insight and knowledge of how a real E-Store like Amazon would be structured.
