Projects

Laptop Deal Finder

This is a webscraper written in Python that looks for gaming laptop on the CanadaComputers website (http://canadacomputers.com/). After scraping the data, a new spreadsheet is generated with current prices, sales, specificaitons such as RAM, CPU and GPU of each laptop. In the future the data will be used for graphs.

Automatic History Deleter

This is an extension written using Javascript, HTML, and CSS that automatically deletes history on startup, that includes certain blacklisted words as a substring, which can be entered in the options section of the chrome extension. There are also a few other functionalities that have been added like the ability to clear history within last minute or last hour.

Spooky Keylogger

This is a windows program written in C that logs keypresses and mouseclicks and then writes them to a log file that gets generated in the same directory, the program is hidden so it can only be found and closed through task manager, in the future I'm planning on adding the ability for the program to transmit the data through the internet.

Comic Book E-Store

This is a fully functional E-Store written in Java, made with a GUI using object-oriented programming practices to simulate how a real E-Store like Amazon would be structured.
